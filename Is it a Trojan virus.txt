Windows erkennt das Programm als Trojaner, dies bitte ignorieren da dies ein Missversändniss des Windows Defenders ist. 

Warum behauptet der Windows Defender, dass das Programm ein Trojaner ist? Es wird ein "Shut-down" command verwendet der den Computer nach den eingegebenen angaben ausschaltet. 
Du kannst im Programm, dann eine 3 stellige Zahl eingeben (Muss nich unbedingt 3 stellig sein) und danach wählst mit den Pfeiltasten aus, ob es sich hierrum um Sekunden, Minuten, Stunden, oder Tage handelt. Wenn du zum beispiel "__3 Minuten" eingibst, dann wird der Computer nach 3 Minuten ausgeschaltet und genau dieser Ausschalt Befehl, wird als Gefahr vom Windows Defender fehlinterprätiert!



Windows recognizes the program as a Trojan, please ignore this as this is a misunderstanding of Windows Defender. 

Why does Windows Defender claim that the program is a Trojan? A "Shut-down" command is used which switches off the computer according to the information entered. 
In the program you can then enter a 3-digit number (doesn't necessarily have to be 3 digits) and then use the arrow keys to select whether it is seconds, minutes, hours or days. For example, if you enter "__3 minutes", the computer will be switched off after 3 minutes and exactly this switch-off command will be misinterpreted as a danger by Windows Defender!
